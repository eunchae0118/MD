교과소개

혁신적GenAI비즈니스응용
교육 목표

1학년 신입생들이 생성형 AI 기술을 이해하고, 이를 통해 미래 비즈니스 환경에서 요구되는 분석 능력과 창의적 문제 해결 능력을 배양하기 위해 설계되었다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • GenAI는 현대 기업들이 더욱 혁신적이고 경쟁력 있는 비즈니스를 운영할 수 있도록 도와준다. 이를 통해 기업은 효율성을 높이고, 고객 경험을 향상시키며, 새로운 시장 기회를 창출할 수 있다. 앞으로 GenAI의 발전은 더욱 다양한 산업 분야에서 핵심적인 역할을 할 것으로 예상된다.
  • 데이터 중심의 비즈니스 의사결정 과정에 있어 AI의 역할을 이해하고, 이를 기반으로 한 전략 수립 능력은 향후 학생들이 졸업 후 경쟁력 있는 전문가로 성장하는 데 필수적이다.
비즈니스인텔리전스와데이터분석실무
교육 목표

비즈니스 인텔리전스와 데이터 분석을 통해 기업의 의사결정을 지원하는 실무 역량을 배양한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • 데이터 수집, 분석, 시각화를 통해 비즈니스 의사결정을 지원하는 실무 능력을 배양한다.
  • 다양한 BI 도구를 활용하여 데이터 기반 인사이트를 도출하는 방법을 학습한다.
Orange3를활용한노코드데이터분석
교육 목표

노코드 도구를 활용한 데이터 분석 역량을 배양하여 비전공자도 데이터를 다룰 수 있게 한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • Orange3 도구를 활용하여 코딩 없이 데이터 분석을 수행하는 방법을 학습한다.
  • 시각적 워크플로우를 통해 머신러닝 모델을 구축하고 평가하는 방법을 익힌다.
컴퓨터는어떻게생각할까? 컴퓨팅사고
교육 목표

컴퓨팅 사고력과 문제 해결 능력을 기르고, 일상의 문제를 컴퓨터 과학적 관점에서 분석하는 능력을 배양한다.

이수 학점

3학점 (이론 2학점, 실습 1학점)

과정 소개
  • 컴퓨터 과학의 기본 원리와 논리적 사고 방법을 학습한다.
  • 알고리즘적 사고와 추상화를 통해 복잡한 문제를 단순화하는 능력을 기른다.
R프로그래밍의이해
교육 목표

R 프로그래밍 언어를 활용한 데이터 분석 역량을 기르고, 통계적 분석 능력을 배양한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • R 언어의 기초 문법과 통계 분석 기법을 학습한다.
  • 데이터 시각화와 통계 모델링을 통해 실제 데이터를 분석하는 능력을 기른다.
프로그래밍 입문
교육 목표

프로그래밍의 기초 개념과 논리적 사고력을 배양하여 소프트웨어 개발의 기반을 마련한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • 프로그래밍의 기본 문법과 알고리즘을 학습한다.
  • Python을 활용하여 간단한 프로그램을 작성하고 문제를 해결하는 능력을 기른다.
공학자를위한C언어기초
교육 목표

C 언어를 활용한 공학 프로그래밍 역량을 기르고, 시스템 프로그래밍의 기초를 마련한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • C 언어의 기초 문법과 공학적 응용을 학습한다.
  • 포인터, 메모리 관리 등 시스템 수준의 프로그래밍 개념을 익힌다.
데이터 분석으로 문제해결
교육 목표

데이터 분석을 통한 문제 해결 역량을 기르고, 데이터 기반 의사결정 능력을 배양한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • 실제 데이터를 활용하여 문제를 분석하고 해결하는 방법을 학습한다.
  • 데이터 전처리, 분석, 시각화의 전 과정을 실습을 통해 익힌다.
디자인소프트웨어AI융합특론
교육 목표

AI와 디자인 소프트웨어의 융합 역량을 기르고, 창의적인 디지털 콘텐츠 제작 능력을 배양한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개
  • 디자인 소프트웨어와 AI 기술의 융합 활용법을 학습한다.
  • 생성형 AI를 활용한 디자인 작업과 창작 프로세스를 실습한다.
의료영상처리 이해
교육 목표

의료 영상 처리의 기초와 응용을 이해하고, 의료 분야의 디지털 기술 활용 능력을 배양한다.

이수 학점

3학점 (이론 2학점, 실습 1학점)

과정 소개
  • 의료 영상의 원리와 디지털 처리 기법을 학습한다.
  • CT, MRI 등 다양한 의료 영상의 특성과 분석 방법을 익힌다.
디자인씽킹과노코딩앱제작(1)
교육 목표

디자인 씽킹과 노코드 앱 개발 역량을 기르고, 사용자 중심의 서비스 기획 능력을 배양한다.

이수 학점

3학점 (이론 1학점, 실습 2학점)

과정 소개

  • 사용자 요구사항 분석부터 프로토타입 제작까지의 전 과정을 실습한다.
디지털 영상정보학
