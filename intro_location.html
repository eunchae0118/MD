서울 방면

• 3번 국도

도봉산역 → 의정부 서부외곽도로 고가 밑 → 의정부 시내 방면 직진 → 롯데아파트 → 망월사역 삼거리 좌회전

• 동부간선도로

수락지하차도 → 도봉차량기지 삼거리 좌회전 → 롯데아파트 삼거리 우회전 → 망월사역 삼거리 좌회전

송추·일산 방면

• 39번 국도

송추검문소 → 경민광장 사거리 우회전 → 서부외곽도로 진입 → 임시 톨게이트 → 도봉동 방면 고가 밑 좌회전 → 망월사역 삼거리 좌회전
